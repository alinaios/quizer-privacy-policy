# AsanaQuiz Privacy Policy

This repository contains the public privacy policy site for the `AsanaQuiz` iOS app.

## Files

- `index.html` — the privacy policy page
- `styles.css` — site styling

## Publish on GitHub Pages

This repo includes a GitHub Actions workflow that deploys the site automatically
when you push to `master` or `main`.

One-time GitHub setup:

1. Push this repository to GitHub.
2. Open the repository on GitHub.
3. Go to `Settings` -> `Pages`.
4. Under `Build and deployment`, choose:
   - `Source`: `GitHub Actions`

GitHub will publish the site at a URL like:

`https://<your-github-username>.github.io/quizer-privacy-policy/`

Use that URL in App Store Connect as the app's Privacy Policy URL.
